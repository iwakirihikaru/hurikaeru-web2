🚀 使い方

最初にやること、APIキー設定、WEB版の配布方法をここにまとめています。

1. 最初の流れ

  1. 下の登録ページを開く
  2. 緑のボタンからスプレッドシートをコピーする
  3. コピー先のシートで初期設定と導入・配布を進める
  4. 終わったらまず先生ページURLから入り、そこから児童ページURLを配布する

2. 先生と児童の入り口

  1. 先生はふだん下の先生ページURLだけを開く
  2. 先生はWEB版の先生ページを使う
  3. 児童には下の児童ページURLだけを配布する
  4. 保存が消えた端末や接続先を入れ直すときだけ接続補助ページを使う
  5. 単元と時間目は先生側の授業スタートに連動する

3. Gemini APIキーの取得

  1. Google AI Studio を開く
  2. API Keys からキーを作成する
  3. 新しく作ったキーをコピーする
  4. 設定タブの「Gemini APIキー」に貼って保存する
新しいキーは auth key として作られます。キーは他人に共有しないでください。

4. APIキーの注意

  • キーはパスワードと同じ扱いです
  • 漏えいしたら新しいキーへ入れ替えてください
  • 動かないときは設定タブで保存し直してください

5. アプリの版情報

  • 版情報の確認は「設定・Aiプロンプト」タブの「版情報」から行います
  • 先生画面では現在の版と配布状況だけを見られます
  • 更新や版戻しは管理者側で行います

ふりかえりアプリ 初期設定ガイド

最初の1回だけ設定が必要です。設定が終われば、ふだんはスプレッドシートを開きっぱなしにしなくても使えます。

1. 最初にやること

  1. 配布ページからテンプレートシートを開く
  2. ファイル -> コピーを作成 を押す
  3. コピーして作った自分のシートを開く
  4. コピーして作ったシートの上にある 導入・配布 メニューを開く
  5. 学校名・学年・組・先生名を入力する
  6. 導入・配布 -> 導入パネルを開く で登録を進め、必要なら Webアプリ化ガイドを開く を押す

2. 大事な注意

  • 配布されたテンプレートは見本です。直接書き込まず、必ずコピーして使ってください。
  • 元テンプレートは閲覧用です。実際に使うのは、自分がコピーして作ったシートです。
  • Google アカウントが複数ログインしているブラウザでは、登録ページや Web アプリが開けないことがあります。開けないときは、まずシークレットウィンドウで試し、それでもだめなときは Edge で開いてください。

3. 導入・配布メニューとは

これは、コピーしたシートをあなた専用のアプリとして使える状態にするための入口です。登録IDの入力は不要です。

次のメッセージが出たら完了です。

  • 登録が完了しました。 と出たら、そのまま 導入・配布 メニューから Webアプリ化 を進めます。

4. 権限確認が出たとき

初回は Google の確認画面が出ることがあります。これは正常です。

  1. 権限を確認 を押す
  2. 自分の Google アカウントを選ぶ
  3. 警告が出たら 詳細 を押す
  4. 無題のプロジェクト（安全ではないページ）に移動 を押す
  5. 許可 を押す

5. 使い始める前の準備

  1. 単元設定 タブで単元を作る
  2. 授業状況 タブで授業を開始する

6. 名簿について

  • 名簿がまだなくても、最初は 1〜40番 で動きます。
  • あとから 名簿 タブで名前を入れれば大丈夫です。

7. ふだんの使い方

設定が終わったら、ふだんはスプレッドシートの中で作業しなくても使えます。

  • じぶんまとめ -> 先生画面を開く
  • じぶんまとめ -> 児童画面を開く

8. 先生画面のタブ紹介

先生画面では、次のタブを使い分けます。最初によく使うのは 単元設定 と 授業状況 です。

📡 授業状況

授業開始、提出状況の確認、児童への一括フィードバック、メダル発表を行う中心の画面です。

📚 単元設定

単元名、教科、時間数、児童に入力させる項目を設定します。最初に使うタブです。

👥 名簿

出席番号と氏名、在籍の有無を整えます。あとからでも更新できます。

🏫 教科デフォルト

教科ごとの定番入力項目を作ります。ここで決めた内容は、新しい単元作成時に引き継がれます。

🤖 プロンプト

AIコメント、AI所見、AI仮評定、メダル人数などを調整します。通常は管理者向けです。

🪵 ログ

AIがうまく動かないときに使う保守用タブです。普段は必要なときだけ見れば十分です。

🧒 ポートフォリオ

児童1人分の記録をまとめて見たり、AI所見の下書きを作ったりできます。

📚 単元記録

クラス全体の記録を振り返る画面です。CSV出力や単元レポート生成にも使えます。

📝 評定する

観点ごとの評定を入力する画面です。根拠を見ながら記録でき、AI仮評定も使えます。

最初によく使う順番

  1. 単元設定 で単元を作る
  2. 授業状況 で授業を開始する
  3. 必要なら 名簿 を整える
  4. 授業後に 単元記録 や ポートフォリオ を見る
  5. 学期末などに 評定する を使う

9. アプリのように使う方法

先生画面を開いたあと、使っている端末に合わせて追加してください。

  • パソコン Chrome: アドレスバー右側の インストール または PCに保存
  • iPad / iPhone: Safari の共有ボタンから ホーム画面に追加
  • Android: Chrome のメニューから ホーム画面に追加 または アプリをインストール

10. スクリーンショットを入れると分かりやすい場所

  • ファイル -> コピーを作成
  • 導入・配布 -> 導入パネルを開く
  • このアプリは Google で確認されていません の画面
  • じぶんまとめ -> 先生画面を開く
  • 授業状況 と 単元設定 の実際の画面

11. うまくいかないとき

  • コピー元ではなく、コピーして作った自分のシートを開いてください。
  • 初期設定 メニューが出ないときは、シートを再読み込みしてください。
  • URL が出ないときは、先に 導入・配布 -> Webアプリ化ガイドを開く から Webアプリ化 をしてください。
  • 児童画面が進まないときは、単元作成と授業開始が終わっているか確認してください。
  • 登録ページや Web アプリが開けないときは、通常ブラウザ → シークレットウィンドウ → Edge の順で試してください。
それでも進まないときは、先生名と登録時刻を管理者へ伝えてください。

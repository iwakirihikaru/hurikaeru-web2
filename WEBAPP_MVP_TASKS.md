# Web/PWA版 MVP タスク分解

## 進め方

- まず `授業開始 -> 児童提出 -> 返却` の一本を通す
- その後に記録一覧、ポートフォリオ、AI補助を足す
- GAS版との完全一致は追わない

## Phase 1: 基盤

- [ ] `webapp/` ディレクトリを切る
- [ ] Next.js を初期化する
- [ ] Supabase プロジェクトを作る
- [ ] `.env.local` のキーを整理する
- [ ] 共通レイアウトを作る
- [ ] PWA 前提のメタ情報を入れる

## Phase 2: 認証と組織

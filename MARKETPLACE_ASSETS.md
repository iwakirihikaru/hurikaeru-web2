# ふりかえりアプリ Marketplace 素材一覧

## 目的

- Google Workspace Marketplace 掲載に必要な素材を抜けなくそろえる。
- 安定版の機能改修と混ぜず、掲載準備を独立して進める。

## 原稿のたたき台

### アプリ名候補

- 第1候補: `ふりかえりアプリ じぶんまとめ`
- 第2候補: `じぶんまとめ`
- OAuth 同意画面、Marketplace 表示名、サポート文言は同じ名称にそろえる
- 50文字以内を維持する

### 短い説明の条件

- [ ] 同じ表現の繰り返しを避ける
- [ ] 何をするアプリかをすぐ伝える
- [ ] 対象ユーザーを含める

### 短い説明 初稿

`小学校の授業で使う、児童のふりかえり記録・教師コメント・単元管理をまとめる Google Workspace 向けアプリです。`

### 詳細説明の条件

- [ ] 導入の流れ
- [ ] 授業中にできること
- [ ] 授業後にできること
- [ ] AI未設定でも使える範囲
- [ ] データの扱い

### 詳細説明 初稿

`ふりかえりアプリ じぶんまとめ は、小学校の授業で使う児童のふりかえり記録を、先生画面と児童画面でまとめて扱える Google Workspace 向けアプリです。先生は単元設定、授業開始、提出確認、コメント返却、メダル付与、単元記録の確認を行えます。児童は共通の児童画面から、その時間の授業に合わせて記録を入力できます。`

`授業中は、先生画面の授業状況から提出状況や記入状況を確認し、必要に応じてコメントの下書きや返却を行えます。授業後は、単元記録やポートフォリオで児童ごとの積み重ねを見返せます。AI を設定した場合はコメント生成や所見作成を補助でき、AI を設定していない場合でも、手入力の下書き保存、返却、手動メダル付与などの基本運用は継続して使えます。`

`このアプリは Google スプレッドシートと Apps Script を利用して動作します。学校や学級ごとの単元、記録、コメントは利用者のスプレッドシートに保存されます。現在は導入時に Google アカウントやブラウザ状態の影響を受けることがあるため、導入ページとガイドで標準の対処手順を案内します。`

### 1文説明 初稿

- 何ができるか
  `授業ごとのふりかえり記録、教師コメント、単元管理をまとめて行えます。`
- 誰向けか
  `小学校の先生と児童向けの授業支援アプリです。`
- Google サービスとの関わり
  `Google スプレッドシートと Apps Script 上で動作します。`
- サードパーティ連携
  `現時点では、主な利用基盤は Google Workspace です。AI補助を使う場合は別途 API 設定を行います。`
- 料金の有無
  `料金有無は未確定のため、公開前に別途整理します。`

## スクリーンショット候補

1. 登録ページ
2. 先生画面 `単元設定`
3. 先生画面 `授業状況`
4. 児童画面
5. `ポートフォリオ` または `単元記録`

## 今はまだ不要なもの

- 大規模な再設計が前提の説明
- 未実装機能の告知
- テスト中導線の説明
- 旧デバッグ表示を含む画像

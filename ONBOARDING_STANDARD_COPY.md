# 導入ページ・ガイド用 標準文言

## 目的

- 登録ページ、配布ページ、初期設定ガイドで使う対処文言を統一する。
- Chrome / Edge / シークレットの切り分け案内を毎回ぶらさない。

## 短い案内文

### 基本導線

`まずはこのページの緑のボタンから進めます。`

### 事前案内

`できれば、使いたい Google アカウントでブラウザにログインし直してから進めてください。`

### コピー先違い

`違う Drive に保存されたときは、そのコピーでは初期設定せず閉じてください。戻って別のアカウントでやり直します。`

## 開けないときの標準対処

### 4行版

1. `まず通常のブラウザで開きます。`
2. `開けないときは、シークレットウィンドウでもう一度開きます。`
3. `それでもだめなときは、Edge で開きます。`
4. `それでもだめなときは、ブラウザの Google ログインを使いたいアカウントだけにしてやり直します。`

### 登録ページ向け

`Chrome で開けないときは、シークレットウィンドウか Edge で開いてください。Google アカウントが複数ログイン中だと、正しいアカウントでも開けないことがあります。`

### Webアプリ化後のURL向け

`作った Web アプリ URL が開けないときは、まずシークレットウィンドウで試してください。通常ブラウザの Google アカウント状態が影響することがあります。`

## ガイドに入れる長めの説明

### そのまま使える文

`Google アカウントが複数ログインしているブラウザでは、正しいアカウントを使っていても、登録ページや Web アプリが開けないことがあります。開けないときは、まずシークレットウィンドウで試してください。それでもだめなときは、Edge など別のブラウザで開いてください。`

### 管理者向け補足

`先生から「現在ファイルを開くことができません」と連絡があった場合は、まずブラウザ依存を疑います。通常ブラウザ -> シークレット -> Edge の順で切り分けると早いです。`

## Marketplace でも使える表現

- `1回の認証で使い始められます`
- `開けないときは別ブラウザでも試せます`
- `初期設定後はふだんスプレッドシートを開きっぱなしにしなくても使えます`

## まだ避ける表現

- `絶対に開けます`
- `どのブラウザでも完全に同じです`
- `自動で全部終わります`
- `デプロイ不要です`（現状はまだ言い切らない）
